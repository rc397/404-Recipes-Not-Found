Contributing

Add a recipe

  1. Create a new HTML file in docs/recipes/.
  2. Start from: Recipe Template
  3. Add your new link to docs/recipes.html.

Style

  • Keep it funny and readable.
  • Stack traces can be fake.
  • Keep it PG-13.
Prefer Markdown instead? You can still write it in Markdown, then we’ll convert it later.

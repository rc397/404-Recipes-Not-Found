# Contributing

## Add a recipe
1. Create a new Markdown file in `docs/recipes/`.
2. Start from the template: [Recipe Template](recipes/recipe-template.md).
3. Add your recipe link to [Recipes](recipes.md).

## Style
- Keep it funny and readable.
- Stack traces can be fake.
- Keep it PG-13.
